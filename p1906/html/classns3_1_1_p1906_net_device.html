Detailed Description

This class implements the network device that will handle all the functionalities conceived within the P1906 framework.

Basically, it is conceived as a container of the communication interface, i.e., the object storing the transmission entity and the reception entity. It enable the interaction between low-layer components of the P1906 framework and upper layers of the protocol stack.
